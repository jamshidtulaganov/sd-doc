| 1C / Esale | двунаправленная | `integration` + `sync` + `api` | [1C / Esale](./1c-esale.md) |

## Общие принципы

- Каждая интеграция пишет в **`IntegrationLog`** (запрос, ответ,
  статус, ошибка). Используйте его как первую остановку при отладке.
- Все исходящие вызовы идут через очередь — никогда inline в обработчике
  запроса.
- Ключи идемпотентности живут на каждой внешней сущности (`XML_ID`, `EXT_ID`).
- Таймауты по умолчанию **15 с**; повторы с экспоненциальной задержкой до 1 часа.
