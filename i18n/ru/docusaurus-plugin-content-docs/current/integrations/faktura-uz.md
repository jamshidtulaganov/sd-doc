# Faktura.uz

Государственный шлюз электронных счетов-фактур в Узбекистане. Отправляет
счета-фактуры с НДС для соответствия требованиям.

## Переключатель

`'enableFakturaUZ' => true` в `params`.

## Поток

1. Заказ достигает `Loaded` (или по правилу тенанта).
2. `FakturaJob` упаковывает счёт-фактуру со строками НДС.
3. Отправляет в Faktura.uz; получает ответ о регистрации.
4. Бухгалтер тенанта подписывает в UI Faktura.uz (EIMZO).
5. Контрагент принимает.
6. Каждый переход зеркалится обратно через polling-задание.

## Распространённые ошибки

| Ошибка | Причина |
|--------|---------|
| Неверный ИНН | `Client.INN` пуст или неверной длины |
| Несоответствие единицы | Единица продукта не в каталоге Faktura.uz |
| Цена > разрешённой по НДС | Проблема конфигурации тенанта |
