# Модуль `store`

:::info Минимальное покрытие
Это страница-заглушка — задокументирован только список контроллеров. Воркфлоу, роли доступа и таблицы БД ещё не описаны. Вклад приветствуется через скилл `sd-docs-author`.
:::

Операции розничного магазина поверх склада — директивы (инструкции
магазинам), закупки и отчёты на стороне магазина.

## Контроллеры

`DirectiveController`, `ListController`, `MaterialReportController`,
`PurchaseController`.
