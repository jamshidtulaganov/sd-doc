# Модуль `gps3`

GPS-трекинг новейшего поколения. Меньше контроллеров (2 / 10
маршрутов), та же схема `d0_gps`, более лёгкий AngularJS-бандл.
Без мониторинга и проигрывания маршрутов — для них пока
используется `gps2`.
