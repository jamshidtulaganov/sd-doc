## Визуальная таксономия для всех flowchart

| Цвет | Класс | Значение |
|--------|-------|---------|
| Синий   | `action`   | Стандартный шаг |
| Янтарный  | `approval` | Требует ревью / одобрения |
| Зелёный  | `success`  | Финальный OK / закрытое состояние |
| Красный    | `reject`   | Финальное состояние с ошибкой / отменой |
| Серый | `external` | Внешняя система (1C, Didox, шлюз, FCM) |
| Фиолетовый | `cron`     | По расписанию / по времени |

(Определено в [Стандартах дизайна воркфлоу](/docs/team/workflow-design).)

## Процедура обновления

Эта галерея генерируется через `sd-docs/scripts/render-diagram-gallery.py`. Запустите повторно после редактирования любого исходника диаграммы на оригинальной странице:

```bash
python3 sd-docs/scripts/render-diagram-gallery.py
```
