# Введение

**SalesDoctor** (внутреннее название **Novus Distribution**, папка кода
`sd-main`) — мульти-арендный CRM для **полевых продаж и дистрибуции**:
ванселлинг, мерчандайзинг, маршрутный учёт.

Эта документация — для **разработчиков**, которые создают, развёртывают
и сопровождают систему. Документация для конечных пользователей
поддерживается отдельно и не входит в этот сайт.

## Что делает система

| Домен | Возможности |
|--------|-------------|
| Продажи | Приём заказов (web + мобильный), цены, скидки, бонусы, возвраты |
| Полевая команда | Маршруты, визиты, GPS, KPI агентов и супервайзеров |
| Мерчандайзинг | Аудиты, фейсинг, опросы, фотоотчёты |
| Каталог | Товары, категории, SKU, бренды, типы цен |
| Склад | Многоскладской учёт, перемещения, инвентаризация |
| Финансы | Дебиторка, оплаты, кассы, P&L по агентам и филиалам |
| Отчёты | 80+ отчётов, дашборды, экспорт в Excel |
| Интеграции | 1C, Didox, Faktura.uz, Smartup, Telegram, FCM, SMS |
| Онлайн | Интернет-магазин, B2B-портал, Telegram-бот |

Полное руководство, ADR и API смотрите в английской версии (раздел
`/docs/`); большинство страниц переводятся последовательно.
