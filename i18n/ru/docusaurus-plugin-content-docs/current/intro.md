# Введение

**SalesDoctor** (внутреннее название **Novus Distribution**, папка кода
`sd-main`) — мульти-тенантная **Distribution CRM**-платформа,
сфокусированная на полевых продажах, ванселлинге, мерчандайзинге и
маршрутном учёте.

Эта документация — для **разработчиков**, которые создают, развёртывают
и эксплуатируют систему. Документация для конечных пользователей
ведётся отдельно и не входит в этот сайт.

## Что делает система

| Домен | Возможности |
|--------|-------------|
| Продажи | Приём заказов (web + мобильный), цены, скидки, бонусы, дефекты, возвраты |
| Полевая команда | Маршруты, визиты, GPS-трекинг, KPI агентов и супервайзеров |
| Мерчандайзинг | Аудиты, фейсинг, опросы, фотоотчёты |
| Каталог | Товары, категории, SKU, бренды, типы цен |
| Склад | Многоскладской учёт, перемещения, инвентаризации, лоты, тара/упаковка |
| Финансы | Дебиторка, оплаты, касса, P&L по агентам и филиалам |
| Отчётность | 80+ отчётов, дашборды, экспорт в Excel |
| Интеграции | 1C, Didox EDI, Faktura.uz, Smartup, Telegram, FCM, SMS |
| Онлайн | Интернет-магазин, B2B-портал, Telegram-бот, отчёты по расписанию |

## Соглашения в этой документации

- Пути к коду указаны относительно корня репозитория (папка `sd-main`).
- `protected/` — корень Yii-приложения.
- Имена модулей пишутся `lowerCamel` и совпадают с папкой, например `orders`,
  `onlineOrder`.
- Роли упоминаются по числовому ID *и* описательному имени, например
  `4 / Agent`.
- Все примеры командной строки рассчитаны на **macOS / Linux** с `bash`.
  В Windows используйте WSL2.
