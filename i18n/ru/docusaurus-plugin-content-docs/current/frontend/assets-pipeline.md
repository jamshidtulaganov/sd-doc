# Пайплайн ассетов (или его отсутствие)

**Build-пайплайна нет**. CSS и JS отдаются как обычные файлы.

## Ментальная модель

- Новый JS / CSS → положите под `js/` или `css/`.
- Сошлитесь из view через `clientScript`.
- Yii копирует всё, что считает «asset bundle», в `assets/<hash>/` в рантайме;
  именно этот URL он подставляет.
- Cache busting — через хешированное имя папки asset'ов.

## Минификация

Опциональна. Большинство файлов отдаются без минификации. Если хотите
минифицировать — запустите свой инструмент локально и закоммитьте оба:
`<file>.js` и `<file>.min.js`, потом ссылайтесь на минифицированный в
прод-сборках.

## Версионирование

Репозиторий не использует frontend-манифест версий. Чтобы форсировать
hard-refresh, добавьте `?v=<release-tag>` во view:

```php
Yii::app()->clientScript->registerScriptFile('/js/orders.js?v=' . VERSION);
```
