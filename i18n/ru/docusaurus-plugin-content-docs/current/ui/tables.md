# Таблицы

Таблицы — рабочая лошадка UI. Построены на DataTables с парой собственных
соглашений.

## Дефолтная таблица

```
┌─────┬───────────────┬──────────┬─────────┬─────────┬────────┬────┐
│  #  │  Order ID  ↕  │ Date  ↕  │ Client  │ Agent   │ Sum  ↕ │ ⋮  │
├─────┼───────────────┼──────────┼─────────┼─────────┼────────┼────┤
│  1  │ O-2026-0123   │ 07-05    │ Acme    │ Ivanov  │  3.2 M │ ⋮  │
│  2  │ O-2026-0122   │ 07-05    │ Beta    │ Petrov  │  1.1 M │ ⋮  │
│ ... │ ...           │ ...      │ ...     │ ...     │  ...   │ ⋮  │
├─────┴───────────────┴──────────┴─────────┴─────────┴────────┴────┤
│  ‹  1 2 3 4 5  ›        Rows per page  [25 ▾]    1–25 of 1247    │
└──────────────────────────────────────────────────────────────────┘
```

## Соглашения

- **Первая колонка** — номер строки (или чекбокс при multi-select).
- **Сортируемые заголовки** помечены `↕`.
- **Денежные колонки** выравниваются по правому краю.
- **Колонки статуса** используют цветные пилюли (зелёная = paid,
  янтарная = pending, красная = problem).
- **Действия строки** живут за меню `⋮` в последней колонке. Избегайте
  inline-кнопок, кроме случаев, когда действие — основной интент.
- **Итоги в футере** для суммы / количества, когда уместно.

## Сгруппированные / раскрывающиеся строки

Используются для иерархических данных (агент → заказы, категория →
товары).

## Производительность

- Server-side пагинация — дефолт.
- Варианты page size: 25 / 50 / 100. Не разрешайте «All» на таблицах,
  которые могут превысить 1k строк.
- Excel-экспорт идёт через экшен **Export**, а не браузерную таблицу —
  никогда не загружайте 50k строк в DOM.
