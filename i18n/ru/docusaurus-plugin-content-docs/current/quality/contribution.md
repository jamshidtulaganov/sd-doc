# Контрибуция

## Воркфлоу

1. **Возьмите тикет** в проектной доске (Jira / Linear / GitHub).
2. **Ветка**: `feat/<short>` или `fix/<short>` от `main`.
3. **Реализация** — держите коммиты маленькими и самообъяснимыми.
4. **Запустите тесты + lint** локально.
5. **Откройте PR** против `main`. Заполните шаблон PR.
6. **Ревью** хотя бы одним maintainer.
7. **Merge** после зелёного CI и одного approval.
8. **Деплой** через релизный поезд.

См. скиллы `claude-ai-assist`, `qa-process`, `pm-workflow` для
AI/QA/PM подфлоу.
