# API v2 — `api2` moduli

Muzlatilgan. Aksariyat kontrollerlar `*.obsolete`; faqat
`MigrateController` texnik xizmat ko'rsatish utiliti sifatida qoladi.

Agar mijoz `/api2/...` ni chaqirayotganini ko'rsangiz, bu eski qurilma.
Uni `protected/modules/api/controllers/ApiLogController.php` orqali
toping va v3 (mobil) yoki v4 (onlayn) ga migratsiyani rejalashtiring.
