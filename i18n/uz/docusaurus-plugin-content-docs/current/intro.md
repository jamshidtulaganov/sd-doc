# Kirish

**SalesDoctor** (ichki nomi **Novus Distribution**, kod papkasi
`sd-main`) — bu **maydon savdosi va distribyutsiya** uchun ko'p
ijarali CRM tizimi: vansel sotuv, merchandayzing, marshrut hisoboti.

Ushbu hujjatlar **dasturchilar** uchun mo'ljallangan. Foydalanuvchilar
uchun hujjatlar alohida saqlanadi va ushbu saytga kirmaydi.

## Tizim nima qiladi

| Soha | Imkoniyatlar |
|------|--------------|
| Savdo | Buyurtma qabul qilish (web va mobil), narxlar, chegirmalar, bonuslar |
| Maydon jamoasi | Marshrutlar, tashriflar, GPS, KPI |
| Merchandayzing | Auditlar, feyseling, so'rovlar, foto-hisobotlar |
| Katalog | Tovarlar, kategoriyalar, brendlar, narx turlari |
| Ombor | Ko'p omborli hisob, ko'chirishlar, inventarizatsiya |
| Moliya | Debitorlik, to'lovlar, kassa, P&L |
| Hisobotlar | 80+ hisobotlar, dashboardlar, Excel eksport |
| Integratsiyalar | 1C, Didox, Faktura.uz, Smartup, Telegram, FCM, SMS |
| Onlayn | Internet-do'kon, B2B-portal, Telegram-bot |

To'liq qo'llanma, ADR va API uchun inglizcha versiyaga qarang
(`/docs/`); ko'pchilik sahifalar bosqichma-bosqich tarjima qilinadi.
