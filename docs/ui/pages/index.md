# UI page reference

One entry per live admin page captured by `scripts/harvest-live.mjs`. 
Currently **72 pages** across **20 modules**, 
from the `admin` role on the demo tenant.

Use this to find:
- the URL of a page,
- which controller renders it,
- what fields, grid columns, and action buttons it exposes,
- and which RBAC permission is required.

Pages are grouped by URL prefix (matching the `protected/modules/<x>` folder in sd-main).
